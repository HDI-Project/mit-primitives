============
Installation
============


Stable release
--------------

To install MIT Primitives, run this command in your terminal:

.. code-block:: console

    $ pip install mit_primitives

This is the preferred method to install MIT Primitives, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From source
------------

The source for MIT Primitives can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/HDI-Project/mit_primitives

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/HDI-Project/mit-primitives/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ make install


.. _Github repo: https://github.com/HDI-Project/mit_primitives
.. _tarball: https://github.com/HDI-Project/mit-primitives/tarball/master
